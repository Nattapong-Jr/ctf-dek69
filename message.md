## Welcom to Git Level

เก่งมากที่ผ่านมาได้ถึงด่านนี้ได้

Git เป็นเครื่องมือที่เราจะได้ใช้บ่อยๆ ทั้งในการเรียนและการทำงานจริง
ด่านนี้เลยถูกสร้างขึ้นมาเพื่อให้ได้ลองใช้งานดู

## Hint

เอาล่ะมาเข้าเรื่องกันคำใบ้ของด้านนี้ก็คือ

```bash
มันอยู่ในอดีต(ต้องย้อนกลับไปดู commit version ก่อนหน้า)
```
